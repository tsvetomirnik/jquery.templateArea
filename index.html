Template Area

Text area with capabilities to define templates

Download Fork Star
Usage
<textarea id="my-textarea">Example {{Placeholder}}</textarea>
 
$('#my-textarea').templateArea({
  placeholders: [
    { name: 'Name', description: 'Full name of the recipient' },
    { name: 'Location', description: 'Address of the recipient' }
  ]
});
Example 1

Only the defined placeholders, visible in the autocomplete list, are considered as valid.

Note: Type "{" to see all available placeholders.

Example 2

Each text wrapped inside {{ }} is considered as a valid placeholder.
